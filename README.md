# GitHub Actions Demo

This project demonstrates a simple CI pipeline using GitHub Actions.

## What it does

- Runs automatically on every push to the main branch
- Sets up Python 3.11
- Installs dependencies from requirements.txt
- Runs automated tests using pytest
- Executes the Python application

## Workflow Steps

1. Checkout Code
2. Set Up Python
3. Check Python Version
4. Show Current Directory
5. List Repository Files
6. Install Dependencies
7. Run Tests
8. Run Python Script

## Technologies Used

- Git
- GitHub
- GitHub Actions
- Python
- Pytest

## What I Learned

I created this project to understand how CI/CD pipelines work in practice. While building it, I learned how GitHub Actions workflows are triggered, how automated tests run using pytest, how to read workflow logs, and how to debug failed pipelines.
